# ShadeAI
An AI like copilot for linux but better
git clone https://github.com/yourusername/shade-ai
cd shade-ai
chmod +x install.sh && ./install.sh

The installer adds shade as a system command automatically.
To add it manually or update an existing install:

sudo cp shade.py /usr/local/bin/shade
sudo chmod +x /usr/local/bin/shade

Then just run: shade
The voice recognition features arent working yet as it needs powerful hardware so to keep things simple just use shade but if u wanna tinker and fix and use it use shade --lt in terminal to enter voice mode.

For more info on what you can do with this AI type /help once you enter shadeAI and it'll help u use it.
